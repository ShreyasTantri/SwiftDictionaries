# Dictionary Practice in Swift

A simple collection of Swift examples showing how to work with dictionaries.

## What’s included

* Basic add, update, and remove operations
* Filtering and searching in dictionaries
* Mapping and reducing values
* Looping through key–value pairs
* Handling dictionaries with array values (like hobbies)

## How to use

Open the playground or Swift file in Xcode and run each snippet to see the output.
